This repo is for doing Frontend Assignment and to submit answer. <br />
Here is the link to the demo -> https://todoapp-testcase.vercel.app/

## Installation

Clone the repo.

```bash
git clone https://github.com/gaisuke/todoapp.git
```

Install the package

```bash
npm install
```

Run project

```bash
npm start
```

## Tech Stack

- ReactJS
- Redux
- TailwindCSS
- Axios
